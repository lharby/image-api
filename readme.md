# Experimental app to test downloading random images via node

## Prerequisites

-   NVM
-   Node (use > v11.0.0)

## Running

Execute `run npm start` in the root folder.

API tokens, secrets, authentication etc are stored in .env file, and should not be pushed to the repo.

Running via sh file

Mac: 
#!/usr/bin/env node

Linux: 
#!usr/local/bin node
